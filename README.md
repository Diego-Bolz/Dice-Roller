# Dice-Roller
A simple interactive dice rolling algorithm developed in python for the terminal.
## Features
- Type of dices: Choose from D2's to D20's
- Amount of dices: Choose how many times the dices must be rolled
- Roll history: Display of all rolls
- Auto calculation: It auto calculates the total value of the dices results plus or minus the modifier
## Technology used
- Python 3
- Native random library
- No generative AI was used
